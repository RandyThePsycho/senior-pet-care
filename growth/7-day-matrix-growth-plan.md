# 7-Day Matrix Growth Plan

This plan starts from the rule in the pasted brief: do not publish until the first planning pass is complete and confirmed.

## Daily Operating Rule

1. Check platform/community rules before the first action of the day.
2. Find more opportunities than we reply to.
3. Reply only when the context is non-emergency, non-diagnostic, and not fresh grief.
4. Log every action in `growth/growth-log.csv`.
5. Log every Feature B/C signal in `growth/needs-intake-log.csv`.
6. Review results against email capture, report opens, journal opens, and 7-day reassessment.

## Day 1 - Readiness And No-Link Trust

Target platforms:
- Reddit: r/seniordogs, r/SeniorCats
- X / Threads

Keywords:
- senior dog good days bad days
- senior pet quality of life checklist
- old dog mobility changes vet questions

Actions:
- Read current subreddit rules.
- Find 10 relevant Reddit posts, reply to 2-3 max with no links.
- Publish 1 no-link educational X or Threads post if not already done for the day.

- Did any phrase repeat across threads?
- Any moderation risk?
