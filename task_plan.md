# Senior Pet Care Growth Execution Plan

## Goal
Build low-risk organic acquisition for Senior Pet Care / PawCheckin while keeping Feature A as the core funnel:

assessment -> email capture -> printable report -> 7-day reassessment -> journal.

Primary metric: 7-day reassessment rate.
Secondary metrics: email capture rate by source, report opens, journal opens, reassessment starts, useful customer language collected.

## Current Phase
Phase 1: careful organic validation through community replies and lightweight channel assets.

## Guardrails
- Do not present the product as diagnosis, treatment, prognosis, or emergency triage.
- Do not post repeated promotional links.
- Prefer useful no-link replies before any link drops.
- Link only when allowed, relevant, and affiliation can be disclosed plainly.
- End-of-life contexts are for listening or gentle support only, not product/tool promotion.
- Do not build Feature B/C before Feature A loop remains healthy.
